p. This is a collection of Unix shell scripts for developers.

p. These scripts should work on all Unix distros and Mac OS X. Otherwise, it will be stated on the file if it doesn't.


h1. SSH-COPY-ID

p. This is for copying your public key to a remote server. Saves you time typing password over and over again.

p. Usage:
p. cd ~/.ssh/ && ssh-copy-id -i id_rsa.pub user@remoteserver

h1. BACKUP MYSQL DATABASES 

1. copy and modify linux/backup_mysqldb.sh
2. ssh to remote server
3. copy contents of file to ~/scripts/backups.sh and chmod a+x ~/scripts/backups.sh
4. crontab -e 
5. Add these if you want to backup database per hour
0 0 * * *  ~/scripts/backups.sh
